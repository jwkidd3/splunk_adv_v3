# Lab 6: Custom Dashboards and Visualizations

**Duration:** 45 minutes
**Difficulty:** Intermediate
**Prerequisites:** Labs 1-5 completed

## Lab Objectives

- Create custom dashboards using Dashboard Studio
- Implement various chart types (line, bar, pie, etc.)
- Add drill-down capabilities
- Use dashboard tokens and inputs
- Convert searches to dashboard panels

## Exercises

### Exercise 1: Create a Dashboard
Navigate to Dashboards > Create New Dashboard

### Exercise 2: Add Visualizations
- Line chart for trends over time
- Bar chart for comparisons
- Pie chart for distributions
- Single value for KPIs

### Exercise 3: Implement Drill-Down
Add click actions to navigate to detailed searches

### Exercise 4: Add Time Range Picker
Create interactive time selection

## Key Topics
- Dashboard Studio vs Classic Dashboards
- Visualization types and use cases
- Dashboard tokens
- Drill-down actions
- Dashboard best practices
