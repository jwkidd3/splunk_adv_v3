# Lab 13: User and Role Management

**Duration:** 30 minutes
**Difficulty:** Intermediate
**Prerequisites:** All previous labs completed

## Lab Objectives

- Create and manage Splunk users
- Configure roles and capabilities
- Implement role-based access control (RBAC)
- Set up authentication methods
- Manage user permissions

## Exercises

### Exercise 1: Create Users
Settings > Access controls > Users > New User

### Exercise 2: Create Custom Role
Settings > Access controls > Roles > New Role

### Exercise 3: Configure Capabilities
Assign specific capabilities to roles

### Exercise 4: Index Permissions
Restrict index access by role

## Key Topics
- User management
- Role hierarchy
- Capabilities and permissions
- Authentication (LDAP, SAML)
- Index-level security
- App-level permissions
